# 手动冒烟验证清单

变更：`add-todo-reminders-and-stats`

## 提醒触发

- [ ] 启动 `pnpm tauri:dev`，确认应用正常进入主窗口。
- [ ] 在主窗口待办中新建一条任务，设置自定义提醒时间为当前时间后 1 分钟。
- [ ] 若系统弹出通知权限请求，选择允许。
- [ ] 等待提醒到期，确认系统通知出现，标题为任务内容，正文包含提醒时间。
- [ ] 回到待办列表，确认同一任务不会重复弹出提醒。
- [ ] 将已触发任务重新设置为 1 分钟后的提醒，确认到期后可再次触发。
- [ ] 将一条任务设为已完成或删除，确认过期后不再触发通知。

## 提醒设置

- [ ] 打开设置面板，进入「提醒设置」。
- [ ] 添加一个快捷提醒选项，修改名称、类型、数值后保存。
- [ ] 回到待办视图，打开提醒下拉，确认新选项立即出现。
- [ ] 点击「恢复默认」，确认二次确认出现，确认后恢复 6 个默认选项。

## 统计视图

- [ ] 点击主窗口侧边栏「统计」，确认主区切换到统计视图且导航高亮。
- [ ] 确认顶部「任务活跃度」热力图渲染，覆盖最近 12 个月。
- [ ] 鼠标悬停有完成记录的日期，确认 tooltip 显示完整日期和完成数量。
- [ ] 确认无完成记录的日期使用最浅色块。
- [ ] 切换系统或应用暗色模式，确认热力图和折线图配色同步变化且文字可读。

## 折线图过滤

- [ ] 在「每日状态趋势」中切换最近 30 / 60 / 90 天，确认图表重新绘制。
- [ ] 依次切换全部、未开始、进行中、暂停、已完成过滤，确认图表重新绘制。
- [ ] 鼠标悬停折线图日期，确认 tooltip 同时展示创建、开始、完成三个数值。

## 重置数据

- [ ] 准备至少一条已完成任务和一条已删除任务。
- [ ] 在统计视图点击「重置数据」，确认二次确认文案包含「永久删除」和「不可恢复」。
- [ ] 点击取消，确认任务数据无变化。
- [ ] 再次点击「重置数据」并确认，确认 toast 显示删除条数。
- [ ] 回到待办视图，确认活动任务保留，已完成和已删除任务被清理。
- [ ] 保持两个窗口打开时执行重置，确认另一个窗口的待办缓存自动刷新。
