# Steno MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现 Steno 跨平台桌面速记 MVP，让用户通过托盘或全局快捷键呼出速记浮窗，自动保存到 SQLite，并可整理为置顶便签、无限画布和 Zen 写作内容。

**Architecture:** Rust/Tauri 负责数据库、备份、窗口、快捷键、托盘和导出命令；Vue 3 负责多视图 UI、Markdown 编辑、自动保存、画布交互和设置体验。SQLite 是本地唯一事实源，所有窗口通过 Tauri commands 读写同一套 `notes` 与 `settings` 数据。

**Tech Stack:** Rust 2024、Tauri 2、tokio、rusqlite、Vue 3、TypeScript、Vite、Pinia、Naive UI、UnoCSS、marked。

import { invoke } from '@tauri-apps/api/core';
import type { Note, SaveNoteRequest } from '@/types/steno';

export function useDb() {
  const saveNote = (input: SaveNoteRequest) => invoke<Note | null>('save_note', { input });
  const listNotes = (limit = 200) => invoke<Note[]>('list_notes', { limit });
  const getNote = (id: string) => invoke<Note | null>('get_note', { id });

  return { saveNote, listNotes, getNote };
}
