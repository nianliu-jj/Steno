常规

开机启动、剪贴板监听、本地数据目录——决定 Steno 在系统里如何驻留。

启动与驻留
开机自动启动

登录后静默进入状态栏，不弹主窗口。

关闭主窗口时隐藏到状态栏

点击红色窗口控件保留后台进程；通过状态栏菜单退出。

浮窗速记球

桌面右下角悬浮捕获入口，参考 MoliTodo 的 Quick Ball。

剪贴板
监听剪贴板变化

自动捕获文本、链接、代码片段，图片单独存档。

最大保留条数

超出后按时间最旧自动清理，Pin 的条目永不清理。

500
自动清理周期

到期后未被 Pin 的剪贴板条目会被清理。

数据
本地数据目录

SQLite 数据库与附件存放位置。修改后需重启。

~/Library/Application Support/Steno
自动备份

每日打包 Markdown 与索引到本地备份目录。

外观

主题、便签纸颜色、Markdown 渲染——让界面贴合你的工作环境。

主题
颜色模式

跟随系统会响应 macOS 外观切换。

主题强调色

影响按钮、链接、Pin 标记。

便签默认底色

新便签的纸张颜色，画布卡片同步使用。

字体与密度
界面字号

影响列表、菜单等系统区域。

13.5px
编辑器字体

Markdown 编辑器与代码块使用。

密度

紧凑模式适合 13" 笔记本；舒适模式更克制。

Markdown 渲染
代码主题

影响 Markdown 代码块高亮样式。参考 milkup 的渲染配色。

预览实时同步滚动

编辑器和预览面板光标位置联动。

快捷键

点击快捷键格子开始录制；按 Esc 取消，Delete 清除。

全局
呼出 / 隐藏浮窗

任何应用内按下即弹出速记面板。

剪贴板历史

打开 EcoPaste 风格的剪贴板检索浮窗。

新建笔记

聚焦主窗口并打开空白笔记。

全局搜索

跨笔记、剪贴板、画布、待办联合检索。

截图与识别
区域截图

调用 snow-shot 风格选区，捕获后进入标注栏。

截图后 OCR

选区结束后自动识别文字到剪贴板。

选中翻译

任意应用内选中文字后翻译到浮窗。

写作
Zen 模式

在编辑器内进入沉浸写作。

切换预览

编辑器单栏 / 双栏 / 仅预览循环切换。

隐私安全

Steno 默认本地优先：所有数据存放在你的设备上，不上传任何服务器。

本地加密
启用数据库加密

SQLCipher 加密本地库；首次启用时设置主密码。

闲置自动锁定

系统闲置达到时长后锁定 Steno，再次访问需要密码。

剪贴板敏感过滤
忽略密码管理器

检测到来自 1Password、Bitwarden 的剪贴板内容时不记录。

过滤疑似敏感内容

信用卡号、API Token、私钥等模式自动跳过。

应用排除名单
这些应用的剪贴板不会被监听

新增后立即生效，参考 tiez-clipboard 的应用白名单实现。

遥测
发送匿名崩溃报告

仅在崩溃发生时上传无标识的栈信息，从不包含笔记内容。
