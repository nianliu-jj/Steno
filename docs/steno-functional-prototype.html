Product architecture

从复制到成文，不切换上下文。

Steno 常驻状态栏，把剪贴板、浮窗速记、置顶便签和无限画布组织成一个本地优先的桌面层。

1. 状态栏守护进程

监听快捷键和剪贴板变化，默认静默，呼出后自动收起。

2. 捕获层

文本、链接、代码片段进入 Inbox，可立即加标签或 Pin。

3. 本地索引

SQLite + Tantivy 风格全文索引，支持时间、来源、标签筛选。

4. 画布层

便签卡片可拖拽、缩放、平移，并可一键智能排列。

5. 写作与导出

Zen 模式收束干扰，最终以 Markdown 导出到本地目录。
