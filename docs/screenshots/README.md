# 应用截图

请在此目录下放置以下截图文件（推荐 16:9 或 4:3 比例，PNG 格式）：

| 文件名 | 说明 |
|--------|------|
| `main-window.png` | 主窗口 — 笔记列表卡片网格 |
| `quicknote-window.png` | 速记浮窗 — 任何应用之上的轻量编辑器 |
| `canvas-view.png` | 无限画布 — 卡片空间化整理 |
| `zen-mode.png` | Zen 写作模式 — 全屏沉浸式编辑器 |
| `settings-panel.png` | 设置面板 — 主题/快捷键/存储路径 |

**截图方法**：
- Windows：`Win+Shift+S` 区域截图
- macOS：`Cmd+Shift+4` 区域截图
- Linux：使用 Flameshot / Spectacle 等截图工具
