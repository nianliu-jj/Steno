本地优先桌面速记

Steno

把闪念、待办和粘贴板内容收进一个本地 Markdown 工作台。快速捕获，稍后整理，默认不离开你的电脑。

查看项目 README 浏览界面截图

为不中断心流的记录设计

Steno 不要求你先选文件夹、打开编辑器、整理标题。它先让内容安全落地，再把整理工作交给主窗口、画布和 Zen 模式。

  • 快速入口 托盘常驻和全局快捷键让速记浮窗随时出现。
  • 本地数据 笔记、待办、粘贴板和设置默认写入本机 SQLite。
  • Markdown 输出 编辑体验接近所见即所得，底层仍是可迁移的 Markdown。
  • 桌面协同 置顶便签、系统通知、本地文件夹和打印窗口都接入桌面环境。

一个窗口，覆盖记录后的整理路径

速记浮窗

从任何应用呼出，输入停顿后自动保存。草稿会回到主窗口，之后可以继续编辑、置顶或归档。

Markdown 写作

ProseMirror 内核保留 Markdown 源码语义，支持代码、公式、图表、图片和大纲跳转。

无限画布

把笔记变成可拖拽卡片，通过空间位置建立上下文，适合会议、项目和灵感整理。

待办和粘贴板

今日待办浮窗、提醒、统计、粘贴板历史和图片编辑补齐桌面工作流中的轻任务。

从捕获到整理，界面保持克制

随时呼出的速记入口

速记浮窗覆盖在当前应用上方，输入内容会自动保存为本地草稿，之后可以继续编辑或转成置顶便签。

空间化整理

无限画布支持平移、缩放和卡片拖拽。双击卡片可以进入 Zen 写作，快速从结构整理切换到长文编辑。

沉浸写作

Zen 模式保留标题、字数、保存状态和大纲，让短笔记可以自然扩展成完整文档。

透明的本地设置

主题、语言、快捷键、提醒、保留天数和数据路径集中在设置面板中管理，所有路径都可以直接查看。

为跨平台桌面应用构建

Steno 的前端负责交互密度和写作体验，Rust 后端负责窗口、快捷键、托盘、SQLite、文件系统和系统能力。

  • 桌面壳 Tauri 2，系统托盘，多窗口，全局快捷键。
  • 前端 Vue 3，TypeScript，Vite，Pinia，Naive UI，UnoCSS。
  • 后端 Rust 2024，rusqlite，tokio，arboard，walkdir。
  • 编辑器 ProseMirror Markdown 内核，代码块内嵌 CodeMirror 6。
  • 渲染 markdown-it，Shiki，KaTeX，Mermaid，DOMPurify。

从一条闪念开始

Steno 适合需要频繁切换上下文的人：开发者、写作者、研究者、会议记录者，以及所有希望把桌面零散输入沉淀为 Markdown 的用户。
