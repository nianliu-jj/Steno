# Semo 高保真响应式原型说明

## 原型范围

本轮交付为一个响应式 Web 产品审核原型，包含入口总览和五个独立屏幕：

- `index.html`：产品总览与评审入口。
- `screens/floating.html`：状态栏常驻与浮窗速记。
- `screens/clipboard.html`：剪贴板历史、搜索、标签与 Pin。
- `screens/canvas.html`：无限画布、拖拽便签、缩放与智能排列。
- `screens/zen.html`：Zen 写作模式、字数统计、本地草稿保存。
- `screens/settings.html`：主题、快捷键、本地优先与导出设置。

## 草图转译

- 草图中的顶部功能搜索栏被转译为桌面端状态栏与产品顶栏。
- 草图中的左侧功能入口被转译为桌面侧栏，移动端改为底部产品导航。
- 草图中的底部快捷入口被转译为浮窗底部操作区和设置页快捷项。
- 草图中的弹窗位置、显示图标和搜索框被合并为可呼出的 Semo 浮窗。

## 视觉系统

选择方向：Human / approachable。

- 背景为干净中性浅底，避免温暖米色或过度渐变。
- 主强调色为绿色，只用于当前状态、主要动作和少量便签状态。
- 面板使用 18 到 28px 圆角，线框和轻微阴影表达浮窗层级。
- 文案全部为中文，适合产品审核时讲清核心能力。

## 已实现交互

- 浮窗呼出、收起、添加速记、置顶笔记。
- 剪贴板搜索、类型过滤、详情预览、Pin 收藏、复制反馈。
- 画布缩放、智能排列、新建便签、拖拽卡片。
- Zen 模式隐藏界面、字数统计、计时、本地草稿保存。
- 设置页主题切换、强调色预览、快捷键录入、开关反馈。
